Key mapping (for Game Boy buttons):
-[Up arrow]: Up
-[Down arrow]: Down
-[Left arrow]: Left
-[Right arrow]: Right
-[E]: A button
-[R]: B button
-[D]: Start button
-[F]: Select button

Options:
-[-]: zoom out
-[+]: zoom in
-[*]: change zoom mode
